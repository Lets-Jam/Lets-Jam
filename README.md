# 딸깍톤 2026 - 딸깍밴드 🎸
> 딸깍톤 프로젝트는 빠른 검증과 구현에 초점을 맞춰 MVP 형태로 진행했습니다. 또한 짧은 개발 기간을 고려해 백엔드와 프론트를 하나의 저장소에서 함께 관리했습니다.

### 개요
딸깍톤에서 진행한 인터랙티브 밴드 프로젝트입니다.
사용자들이 각자 역할(보컬, 악기 등)을 맡아 동시에 참여하여 하나의 음악을 완성하는 실시간 협업 서비스입니다.

### 주요 기능
- 실시간 멀티 유저 음악 연주
- 역할별 파트(보컬, 기타, 드럼 등) 참여
- 동기화된 재생을 통한 합주 경험 제공

- 디자이너: 김장현
